Politique de Confidentialité

Votre confiance est notre priorité. Découvrez comment nous protégeons vos données.

1. Introduction

Chez Agent Boutique, nous nous engageons à protéger la vie privée de nos utilisateurs et à garantir la sécurité de leurs données personnelles. Cette politique de confidentialité explique quelles informations nous collectons, comment nous les utilisons et avec qui nous les partageons.

Dernière mise à jour : 1er Janvier 2026
Cette politique s'applique à tous les services d'Agent Boutique et à nos utilisateurs.

2. Données que nous collectons

2.1 Données personnelles

Nous collectons les informations suivantes lorsque vous utilisez nos services :

  • Nom et coordonnées : Nom, prénom, adresse email, numéro de téléphone
  • Informations professionnelles : Nom de l'entreprise, poste, secteur d'activité
  • Données de connexion : Adresse IP, type de navigateur, système d'exploitation
  • Données d'utilisation : Pages visitées, temps passé, fonctionnalités utilisées

2.2 Données de communication

Pour fonctionner, notre service traite les communications WhatsApp :

  • Messages échangés avec vos clients
  • Métadonnées des communications (horodatage, expéditeur, destinataire)
  • Informations de qualification des prospects

3. Utilisation des données

Nous utilisons vos données pour les finalités suivantes :

Finalité Données concernées Base légale
Fournir le service Toutes les données collectées Exécution du contrat
Améliorer le service Données d'utilisation Intérêt légitime
Support client Données personnelles Intérêt légitime
Marketing Coordonnées Consentement

4. Partage des données

4.1 Sous-traitants

Nous partageons vos données avec des prestataires de confiance pour :

  • Hébergement : AWS pour l'infrastructure cloud
  • Communications : Meta/WhatsApp pour l'envoi de messages
  • Paiements : Stripe pour le traitement des transactions
  • Analyse : Google Analytics pour les statistiques d'utilisation

4.2 Transferts internationaux

Certains de nos sous-traitants sont basés hors de l'Union Européenne. Nous nous assurons que des garanties appropriées sont en place (clauses contractuelles types, certifications adéquates).

5. Sécurité des données

Nous mettons en œuvre des mesures techniques et organisationnelles robustes pour protéger vos données :

  • Chiffrement : Données chiffrées au repos et en transit (TLS 1.3)
  • Contrôle d'accès : Authentification multi-facteurs et principe du moindre privilège
  • Surveillance : Détection d'intrusions et audits de sécurité réguliers
  • Sauvegardes : Sauvegardes chiffrées et stockées géographiquement redondantes
Certification : Nos processus sont conformes au RGPD et nous travaillons vers une certification ISO 27001.

6. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : Savoir quelles données nous avons sur vous
  • Droit de rectification : Corriger les données inexactes
  • Droit d'effacement : Demander la suppression de vos données
  • Droit de limitation : Limiter le traitement de vos données
  • Droit de portabilité : Recevoir vos données dans un format lisible
  • Droit d'opposition : Vous opposer au traitement pour certains motifs

Pour exercer ces droits, contactez-nous à : Contact sur demande

7. Conservation des données

Nous ne conservons vos données que le temps nécessaire aux finalités pour lesquelles elles ont été collectées :

  • Données clients : Pendant toute la durée du contrat et 7 ans après
  • Données prospects : 3 ans maximum sans contact
  • Données techniques : 13 mois maximum
  • Données marketing : Jusqu'à désabonnement

8. Cookies et technologies similaires

Nous utilisons des cookies pour améliorer votre expérience :

  • Cookies essentiels : Nécessaires au fonctionnement du site
  • Cookies de performance : Pour analyser l'utilisation du site
  • Cookies de fonctionnalité : Pour mémoriser vos préférences

Vous pouvez gérer vos préférences cookies via notre panneau de configuration ou les paramètres de votre navigateur.

9. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité pour refléter des changements dans nos pratiques ou pour des raisons réglementaires. Toute modification sera notifiée par :

  • Email aux utilisateurs concernés
  • Notification sur le site web
  • 30 jours avant l'entrée en vigueur pour les changements significatifs
